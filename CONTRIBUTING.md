# Contributing

Improvements welcome. Useful contributions:

- **New starter questions** for domains the template covers thinly (own wording only — see the licensing rule below)
- **Better worked examples** — especially Pattern 3 (gap + compensating control) examples from other domains
- **Corrections** to anything inaccurate
- **Translations** (Canadian French particularly welcome)

## Rules

1. **No licensed question text.** SIG/SIG Lite (Shared Assessments) and CAIQ (Cloud Security Alliance) question sets are licensed works. Submissions must use original wording. PRs containing official question text will be declined.
2. **Keep answers honest in spirit.** Worked examples must model truthful answering — no "how to make a no sound like a yes."
3. **One concern per PR**, with a short description of what it improves.

## How

Fork → branch → PR. For anything substantial, open an issue first so we don't duplicate effort.
